# CipherBoard Word Lists v1

`words_en_v1.txt` and `words_ru_v1.txt` are protocol data. Their line numbers,
starting at zero, are the 12-bit values used by the `CBW1` presentation codec.
Published v1 lists must never be reordered or edited in place.

The word codec is only a reversible presentation of the authenticated `CB1`
envelope. It does not change the cryptography and is not steganography: a
uniform sequence of unrelated words remains distinguishable from natural
prose. Decoding is exact and does not autocorrect, normalize, or guess words.

The lists are reproducibly derived from
[`hermitdave/FrequencyWords`](https://github.com/hermitdave/FrequencyWords),
commit `525f9b560de45753a5ea01069454e72e9aa541c6`, files
`content/2018/en/en_50k.txt` and `content/2018/ru/ru_50k.txt`. The source data is
licensed under CC-BY-SA-4.0. The repository carries the license as
`LICENSE-CC-BY-SA-4.0`.

Selection preserves source frequency order, removes duplicates, accepts only
lowercase alphabetic words of four through ten characters, excludes Russian
`ё`, and applies the explicit and fragment deny-lists in
`generate_wordlists.ps1`. Those deny-lists remove profanity, explicit sexual
terms, slurs, and high-impact terms related to violence, weapons, drugs, and
terrorism. Exact forms are used where a broad substring would remove ordinary
words (for example, Russian `кровать`). This is a best-effort content filter,
not a linguistic safety guarantee.

Run `generate_wordlists.ps1`, review samples and the deny-list scan, then run
the Rust tests. Tests pin the generated byte hashes and all structural
invariants.
